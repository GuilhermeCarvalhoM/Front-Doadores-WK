Bem-vindo ao Sistema de Processamento de Dados para Banco de Sangue!

Nosso sistema foi projetado para ajudar uma agência de banco de sangue a processar e extrair informações importantes de uma lista de candidatos a doadores. Ao fornecer um arquivo JSON com os dados dos candidatos, nosso sistema apresentará os seguintes resultados:


Distribuição dos Candidatos por Estado

Alguns insights sobre a distribuição dos candidatos por estado:

  • Minas Gerais (MG) possui 8 candidatos.
  • Sergipe (SE) possui 22 candidatos.
  • Amazonas (AM) possui 5 candidatos.


A quantidade de possíveis doadores para cada tipo sanguíneo receptor

Insights sobre Doadores de Sangue

  • Distribuição dos tipos sanguíneos: O tipo sanguíneo O- possui a maior quantidade de doadores disponíveis, seguido por A-, B-, O+, A+, B+, AB- e AB+.
  • Escassez de doadores de AB+: O tipo sanguíneo AB+ é o que possui a menor quantidade de doadores disponíveis, com apenas 18 indivíduos listados.
  • Alta disponibilidade de doadores de O-: O tipo sanguíneo O- possui uma quantidade significativa de doadores disponíveis, com 190 indivíduos listados.
  • Variação na quantidade de doadores por tipo: A quantidade de doadores varia consideravelmente entre os diferentes tipos sanguíneos.
